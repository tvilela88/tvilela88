## Featured Project: Raiza

**AI-Powered Recruitment Platform** — a full-stack SaaS application that automates candidate sourcing, CV parsing, job matching, and application tracking.

### What it does
- **Conversational Lead Conversion** — 11-phase AI-driven funnel that qualifies leads and onboards clients through natural conversation
- **WhatsApp Bot Integration** — collects CVs and candidate data via automated WhatsApp conversations
- **AI CV Parsing** — extracts structured data from PDFs, images, and documents using multi-provider AI (OpenAI, Gemini, Claude)
- **Intelligent Job Matching** — scores candidates against job requirements using configurable AI-powered matching with transit time calculations via Google Maps
- **Real-Time ATS** — Kanban-style application tracking with live updates via WebSockets
- **Credits System** — prepaid credit model with AI-powered contestation handling
- **Job Image Generation** — auto-generates branded job posting images using Puppeteer

### Architecture
- **Backend**: Node.js + Fastify + TypeScript, PostgreSQL with PostGIS, Redis for caching/sessions
- **Frontend**: Next.js 14 + React + Tailwind CSS + Zustand, real-time updates via Socket.IO
- **Infrastructure**: Docker, GitHub Actions CI/CD, Traefik reverse proxy, MinIO object storage
- **AI Layer**: Multi-provider (OpenAI, Google Gemini, Anthropic Claude) with fallback chains
